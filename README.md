## 安装教程  
- 第一步，安装油猴插件：[Tampermonkey](https://www.tampermonkey.net)
- 第二步，安装数码小站脚本：[数码小站](https://greasyfork.org/zh-CN/scripts/426806)  
- 暴力猴可能存在卡密一直查询中的问题
- 也可以使用[KinhDown客户端](https://kinhdown.com)，可批量下载
